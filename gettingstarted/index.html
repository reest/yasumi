Getting Started

Using Yasumi in your projects is super easy! Yasumi is available on Packagist and can be installed using Composer.

System Requirements

You need PHP >= 7.1 to use azuyalabs/yasumi but the latest stable version of PHP is recommended. Yasumi is verified and tested on PHP 7.1, 7.2 and 7.3. Other versions of PHP are no longer supported.

In case you are still using the previous major release of Yasumi (v1.8.0) with PHP 5.x, a bugfix release has been made available for the v1.8 branch. You can read more about it here: Welcome Yasumi version 2!. However it is strongly recommended to upgrade to PHP 7 and Yasumi v2.0.0.

Installation

Install azuyalabs/yasumi using Composer by issuing the following command:

$ composer require azuyalabs/yasumi

Be sure to also include the Composer autoload file in your project:

<?php

require 'vendor/autoload.php';

// Use the factory to create a new holiday provider instance
$holidays = Yasumi\Yasumi::create('USA', 2019);

That's all it takes, you're ready to go! Have a look in the Cookbook to see how you can use Yasumi.
